# SuddenMathProblem

`SuddenMathProblem`은 Paper 서버에서 동작하는 플러그인입니다.
현재 버전(1.0)은 **기본 플러그인 뼈대만 포함**되어 있으며, 서버에서 로드/언로드되는 기본 동작만 제공합니다.

## 플러그인 기능
- 서버 시작 시 플러그인이 활성화됩니다.
- 서버 종료(또는 플러그인 비활성화) 시 플러그인이 비활성화됩니다.
- 현재 버전에는 문제 출제, 점수 기능, GUI 등 추가 게임 기능이 구현되어 있지 않습니다.

## 사용 대상 및 필요 환경
- 서버 종류: **Paper**
- 지원 API 버전: **1.21**
- 플러그인 버전: **1.0**

## 인게임 명령어
현재 버전에서 제공되는 인게임 명령어는 없습니다.

- `/suddenmathproblem` 같은 전용 명령어: **미지원**
- 권한(permission) 기반 명령어: **미지원**

## config 파일 설정
현재 버전에서는 `config.yml`이 제공되지 않으며, 유저가 조정할 수 있는 런타임 설정값이 없습니다.

### 현재 메타 정보 파일
- `src/main/resources/paper-plugin.yml`
  - `name`: 플러그인 이름
  - `version`: 플러그인 버전
  - `main`: 메인 클래스 경로
  - `api-version`: 대상 API 버전

위 파일은 플러그인 메타데이터이며, 일반적인 게임 내 동작 옵션(`시간`, `난이도`, `보상` 등)을 조정하는 config 파일은 아직 없습니다.

## 설치 방법
1. 빌드된 플러그인 JAR 파일을 서버의 `plugins/` 폴더에 넣습니다.
2. 서버를 재시작합니다.
3. 서버 로그에서 플러그인이 정상 로드되었는지 확인합니다.

---
추후 실제 게임 기능(문제 출제/정답 처리/보상 등)이 추가되면, 이 문서에 인게임 명령어와 `config.yml` 설정 항목을 업데이트할 수 있습니다.
