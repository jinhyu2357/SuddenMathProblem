# SuddenMathProblem

`SuddenMathProblem`은 Paper 서버에서 동작하는 플러그인입니다.

## 명령어
- `/mathproblem` : op권한을 가진 플레이어가 강제로 문제를 제시할 수 있습니다.
- `/mathproblemreload` : config파일을 다시 불러온다.
- `/mathpass` : 현재 수학 문제를 패스합니다.

## config 파일 설정
플러그인 폴더에서 값을 수정할 수 있습니다.
- 정답 입력시 발생하는 효과음&폭죽의 여부를 on/off할 수 있습니다.
- 정답 입력시 실행되는 명령어를 수정할 수 있습니다.
- 출력 메세지 내용을 수정할 수 있습니다.
- 문제에 사용되는 '문제 출제 주기' & '최대 숫자' 등을 수정할 수 있습니다.
